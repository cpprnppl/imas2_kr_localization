
# IM@S2 Font Auto Patcher 
  
아이돌마스터2(PS3/RPCS3) 폰트 리소스(NFH/NUT)에 한글 글리프를 추가하기 위한 **Replace-only** 패처입니다.   
텍스처 슬롯을 추가(add)하지 않고, 기존 한자 글리프 위치(희생 슬롯)에 한글을 덮어써서 안정적으로 동작하도록 설계했습니다.  
  

  
## 특징  
  
- Drag & Drop으로 `.nut` 파일을 넣으면 같은 폴더의 `.nfh`를 자동으로 매칭  
- 원본은 `_backup/` 폴더에 타임스탬프 붙여 자동 백업  
- 패치 결과는 **원본 파일명 그대로 덮어쓰기** (게임 폴더에 바로 투입 가능)  
- 문자셋은 `korean_2350.txt`(2350 기본 문자) + 사용자 추가 입력  
- NotoSansKR Medium 기준 기본값:  
 - `font_size=26`, `baseline=20`, `x_offset=0`, `row_h=29`  
- 설정/최근 파일 경로 자동 저장 (재실행 시 자동 로드)  
- 폰트 매핑(`font_mapping.json`) 자동 저장:  
 - **선택한 폰트 파일(.ttf/.otf)과 동일 폴더**에 생성됨  
- fixed-slot 타입 NUT(예: `im2_font_l`)에서 뷰어가 슬롯 크기를 이상하게 표시하는 문제를 줄이기 위해  
 헤더 필드를 정규화(파일 크기/stride는 유지)  
  

  
## 준비물  
  
- Python 3.10+ 권장 (3.14도 가능)  
- 의존성 설치

    - pip install freetype-py numpy tkinterdnd2




## 파일 구성

-   `patcher_auto.py` : GUI 패처 스크립트
    
-   `korean_2350.txt` : 기본 한글 2350 문자셋 (스크립트와 같은 폴더에 둡니다)
    
    


## 사용법

1.  GUI에서 한글 폰트 파일(.ttf/.otf)을 선택합니다. (예: NotoSansKR-Medium.otf)
    
2.  게임의 폰트 NUT 파일을 드래그&드롭합니다.
    

-   예: `im2_font.nut`, `im2_font_l.nut`
    
-   같은 폴더에 동일 이름의 `.nfh`가 있어야 합니다.
    

3.   **원본 파일명 그대로 패치가 진행됩니다**
    
-   원본 백업: `<폴더>/_backup/` (타임스탬프 포함)
    
-   텍스트 매핑 테이블: `<폰트폴더>/font_mapping.json` (추후 번역본 스크립트 변환시 사용)
    

## 문자셋(한글) 구성

-   기본 문자셋: `korean_2350.txt`
    
-   추가 문자: GUI의 “추가할 글자” 입력란에 입력(공백/개행 무시)
    
-   최종 문자셋은 “기본 + 추가”로, 자동 중복 제거 후 패치가 진행됩니다.
    


## 폰트 매핑: `font_mapping.json`

Replace-only 방식이므로, 게임 텍스트에 한글을 직접 넣는 것이 아니라  
**한글 → 희생 한자 코드로 치환**한 문자열을 게임 데이터에 넣는 것을 전제로 만든 패치입니다.

`font_mapping.json`은 다음 형태입니다.

{  
 "가": "籀",  
 "각": "籃"  
}

즉, 텍스트에 `"가"`를 넣고 싶으면 실제 파일에는 `"籀"`가 들어가야 게임에서 `"가"`로 렌더링됩니다. 따라서 csv, xlsv등 번역된 스크립트를 변환할 때 이 json을 참조하여 치환해야 합니다.



## 튜닝 팁 (baseline)

한글이 다른 글자보다 위/아래로 치우치면 `baseline` 값을 조절하세요.

-   baseline 값을 **낮추면** 글자가 **위로** 올라갑니다. (예: 20 → 18)
    
-   baseline 값을 **올리면** 글자가 **아래로** 내려갑니다. (예: 20 → 22)
    



## 주의사항

-   Replace-only 방식이므로 “희생할 한자 슬롯”이 충분해야 합니다. 
    
-   보호 한자 리스트(JIS 레벨1)는 기본으로 적용되어 있어, 자주 쓰는 한자는 최대한 보존됩니다.
